
# `lib/pages/sns_page.dart`

`sns_page.dart`は、SNS（ソーシャルネットワーキングサービス）の機能を模したUIを提供し、投稿のリストを表示します。

## 主な機能

- **投稿リストの表示:**
  - `ListView.builder`を使用して、`PostCard`ウィジェットのリストを動的に生成し、スクロール可能な投稿フィードを作成します。

- **投稿カード (`PostCard`):**
  - 各投稿には、ユーザーのアバター、名前、投稿画像、およびインタラクションボタン（いいね、コメント、シェア）が含まれています。
  - `Card`ウィジェットでラップされており、視覚的に区別されたコンテナを提供します。
  - ユーザー情報は`UserInfo`セクションに表示されます。
  - 投稿画像は`PostImage`セクションに表示されます。
  - いいね、コメント、シェアボタンは`PostActions`セクションに配置されます。
