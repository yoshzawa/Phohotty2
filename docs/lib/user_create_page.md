## `pages/user_create_page.dart`

- **目的**: 新しいユーザーアカウントを作成するための画面です。
- **主要機能**:
    - メールアドレスとパスワードによるユーザー登録フォームを提供します。
    - 表示名、メールアドレス、パスワード（確認用含む）を入力し、バリデーションチェックを行います。
    - `FirebaseAuth`を利用して、新しいユーザーアカウントを作成し、表示名を更新します。
    - Googleアカウントを使用したサインイン（および新規登録）機能も提供します (`FbAuth.instance.signInWithGoogle`)。
    - フォームの送信時やGoogleサインイン時には、ローディングインジケーターを表示します。
    - ユーザー作成後、前の画面に自動的に戻ります。
