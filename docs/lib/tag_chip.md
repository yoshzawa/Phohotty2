## `widgets/tag_chip.dart`

- **目的**: タグを表示するためのカスタムUIウィジェットです。
- **主要機能**:
    - `label`: 表示するタグのテキストです。
    - `selected`: タグが選択されているかどうかを真偽値で示します。
    - `custom`: AIが提案したタグか、ユーザーが独自に追加したカスタムタグかを区別します。
    - `onTap`: タグがタップされたときのコールバック関数です。
    - 選択状態やカスタムタグかどうかによって、背景色、ボーダー色、アイコン、テキスト色が動的に変化します。
    - 英数字のみで構成されているかどうかに基づいて、自動的にカスタムタグ（日本語など）を判別するロジックも含まれています。
